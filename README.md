# Network Topology Simulator

Welcome to the **Network Topology Simulator**, a web-based interactive tool designed to help users **visualize, design, and simulate computer networks**.  
It allows you to create routers, switches, and hosts, connect them dynamically, and observe how packets flow through the network in real time.

---

## Overview

The simulator provides an intuitive **drag-and-drop canvas interface** for building and experimenting with network topologies.  
Users can easily visualize how devices interconnect and explore how routing works between nodes.
